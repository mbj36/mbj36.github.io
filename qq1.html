How Can I Fully Utilise 2 Months Of Summer Break?I want To Learn Coding.Should I Start With HTML,CSS Or go With Python/PHP Or Django?
Yes learn Python this summer. And you can do it on your own too. It is pretty easy to get started with. If you are indeed going to learn Python, I would recommend you take a couple of days to get acquainted with the basics. 3 - 4 days max. After that, you should try Django or Flask. I see in the question you've put "Python/PHP/Django" together. You should really try to understand the differences between them. While Python and PHP are both programming languages, Django is a web framework for Python, and a really good one to start with. So, coming back to Django or Flask, I would recommend you choose Django over Flask. While Flask is a minimal and super light weight framework, Django is more like a monster. It will give you power in developing you web app. Head over to http://djangoproject.com/ and read and complete the tutorial. You may not complete the entire tutorial, but upto part 4, if I recall correctly should get you up to speed. This might take you around 2 days. Try to understand the concepts clearly instead of just blindly copying code. In case you are stuck, try asking around on Stackoverflow or IRC. Once you are through this, make something of your own. The ones I can think of are: 1. TODO App 2. A command line paste utility. Try including Bootstrap for your designing, and learn and use some Javascript/Jquery/AJAX. Will update if I can think of something else. Once you are done through these, try your hand in writing a web crawler. It will teach you a lot. Most importantly, try finding where your interest lies. I for instance, like to build Network Applications. And keep thinking of things you really implement. These could be ones already made. But its important to keep making something. Push all your code to Github. You say you want to contribute to Open Source. Browse Github for interesting projects. Talk to the developers involved. They will be delighted to have a new contributor and they will help you get started. If you are aiming for Google Summer of Code 2015, head over to program page, and browse through the list of organizations accepted this year and also the ones in previous years. Search with tags like Python. Find one that interests you a lot, and again talk to them. Start reading their code, find an issue you think you can solve and get dirty. If you have any queries, feel free to inbox me. I will be glad to help you out. Thanks for the A2A! Happy coding! :)
